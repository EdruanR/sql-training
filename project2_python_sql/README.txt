# Proyecto 2 - SQL + Python (Semana 2)

Este proyecto combina **PostgreSQL** con **Python (Pandas, SQLAlchemy, Matplotlib)** 
para realizar un análisis de datos usando la base de ejemplo **Chinook**.

## Contenido
- **Conexión a PostgreSQL** desde Python.
- Consultas SQL ejecutadas directamente desde Jupyter Notebook.
- Análisis de:
  - Ventas totales por país.
  - Evolución mensual de las ventas.
  - Ranking de los 10 clientes más valiosos.
- Mini-dashboard con gráficas.

## Requisitos
- PostgreSQL 17 (con base Chinook cargada).
- Python 3.13, Anaconda.
- Librerías: pandas, sqlalchemy, psycopg2, matplotlib, jupyter.

## Cómo usar
1. Abrir el notebook `analysis.ipynb`.
2. Seleccionar el kernel de Anaconda (`sql_training`).
3. Ejecutar las celdas secuencialmente para reproducir queries y gráficos.
