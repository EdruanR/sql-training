# Mini Proyecto SQL - Chinook DB

Este proyecto corresponde a la **Semana 1** de mi entrenamiento en SQL.
El objetivo fue practicar consultas básicas usando el dataset **Chinook** en PostgreSQL.

## Contenido
- Exploración de tablas principales ('Customer', 'Invoice', 'Track', 'InvoiceLine').
- Consulta de **Top 10 productos más vendidos**.
- Identificación de **principales clientes por producto** usando CTEs y JOINs.

## Cómo usar
1. Crear la base de datos 'chinook' en PostgreSQL.
2. Importar el script 'Chinook_PostgreSql.sql'.
3. Ejecutar los queries ubicados en la carpeta 'sql/'.

## Resultado
Ejemplo de salida:  
'track_name | customer_name | country | purchases_by_customer | rank_customer'
